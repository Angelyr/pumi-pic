5 /*
6  Provides a timing utility to record and output timing of operations.
7 
8  The output can be controlled using this function:
9  SetTimingVerbosity(verbosity)
10  Details of each level of verbosity can be found below
11 
12  By default the timing functions record on process 0 of MPI_COMM_WORLD.
13  Recording can be enabled and disabled on each process using these functions:
14  EnableTiming() - enable calling process to record timing
15  DisableTiming() - disable calling process to record timing
16 
17  To record timing of an operation use:
18  RecordTime(string, seconds, prebarrierTime (optional))
19  This will accumulate all calls with the same `string` and if verbosity is set high enough print a message with the provided timing
20 
21  To print the accumulated timing information you can call either:
22  SummarizeTime() - prints timing info for enabled processes
23  SummarizeTimeAcrossProcesses() - prints averaged timing info over all enabled processes
